# DEVERSE

<strong>Descrição do Projeto</strong>

<ul>Este projeto consiste no desenvolvimento de um site que reúne códigos prontos de interface, organizados por categorias, com foco educacional. A plataforma foi criada com o objetivo de facilitar o aprendizado e a reutilização de componentes visuais comuns em aplicações web.

O site disponibiliza exemplos funcionais como botões com efeito hover, animações em CSS, loaders, checklists interativas, formulários e outros elementos de interface. Todos os códigos foram estruturados para serem simples, claros e fáceis de entender.
</ul>

<strong>Objetivos</strong>
<ul>
Organizar componentes de interface por categorias

Praticar estruturação de páginas com HTML semântico

Aplicar estilização moderna com CSS

Implementar interações utilizando JavaScript

Criar uma interface com tema escuro e destaque em verde neon

Desenvolver organização de arquivos e documentação técnica
</ul>

<strong>Funcionalidades</strong>

<ul>Página inicial com apresentação do projeto

Página de categorias organizada em grid

Cards com descrição de cada tipo de componente

Exemplos visuais funcionais

Botões com efeito hover

Animações CSS

Loaders animados

Checklists interativas

Código comentado para facilitar o entendimento

Botão para copiar código

Layout responsivo</ul>

<strong>Tecnologias Utilizadas</strong>

<ul>HTML

CSS

JavaScript</ul>

<strong>Recursos aplicados:</strong>

<ul>Flexbox e Grid Layout

Pseudo-classes como :hover

Animações com @keyframes

Manipulação básica de DOM

Design e Identidade Visual</ul>

<strong>O site utiliza:</strong>

<ul>Tema escuro como base principal

Verde neon como cor de destaque

Tipografia moderna

Interface minimalista focada em leitura e funcionalidade

O uso do verde neon foi aplicado principalmente em botões, links ativos e estados de interação, evitando excesso visual.</ul>

<strong>Como Executar o Projeto</strong>

<ul>Faça o download ou clone do repositório.

Abra a pasta do projeto.

Execute o arquivo index.html em um navegador.

Opcional:
Utilizar a extensão Live Server no VS Code para melhor experiência de desenvolvimento.</ul>
